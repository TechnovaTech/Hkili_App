# Story Mode Images

Place the following image files in this folder:

- i1.png - Vegetable/Inventors theme image
- i2.png - Environment/Construction site theme image  
- i3.png - Jungle Book theme image
- i4.png - Alice in Wonderland theme image
- i5.png - Grimm's Tales theme image
- i6.png - Wizard of Oz theme image

Images should be:
- PNG format
- Recommended size: 300x300 pixels or larger
- Kid-friendly illustrations matching each story theme
